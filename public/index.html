Local Conversation Prototype

Chat with local files, conversation memory, and retrieval rules.

This app uses local content and optional no-key web retrieval only. It does not call OpenAI or paid AI services, does not require an API key, and does not claim to be a trained hosted LLM. It searches saved local/web knowledge before responding.

    Training

    Local model training

    Put .txt, .md, or .json files in local_training_data/, then train the existing transformer locally for next-token prediction. Training may be slow on CPU. Better results require more local text, a suitable model size, more training time, and available CPU/GPU.

    This does not create a ChatGPT-quality model, does not make the model intelligent, and it is not fully trained unless a checkpoint exists.
