This app keeps the visible interface focused on normal chat and article/webpage handling. It does not call OpenAI or paid AI services, does not require an API key, and only runs user-triggered web search when the exact phrase “search the internet for” is present.

Summarize or inspect a page

Paste a URL in chat with a request like summarize https://example.com/article or ask for a specific detail from that URL. Direct URL retrieval is separate from internet search. Web search only runs when your prompt includes the exact phrase search the internet for.

  • Summaries include a short summary, key points, important details, names/dates/numbers, and source URL.
  • Specific-detail questions search within the retrieved page text and answer from that content.
  • No OpenAI, paid services, or API keys are required.
